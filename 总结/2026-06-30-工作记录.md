# 工作记录 — 2026-06-30

## 项目
沪深300指数预测（THU-BDC2026 大数据竞赛）

## 问题诊断

### 现象
改进版模型（V6）`final_score` 翻倍（0.038→0.077），但实际收益率从基准的 2.52% 降至 0.46%。

### 根因分析（三项独立调查）

1. **标签错配**：`_build_label_and_clean` 将绝对收益转换为超额收益（减当日市场均值），模型学习"谁比市场平均好"而非"谁涨得多"。牛市全涨时没问题，市场震荡/下跌时选出的是"跌得少"而非"涨得多"的股票。

2. **配置自毁**：V6 同时做了四件互相放大危害的事：
   - 模型缩容（128维/2层）→ 只能学简单动量信号
   - 关闭辅助任务 → 失去风险感知正则化
   - 关闭数据增强 → 对动量模式过拟合
   - 加 fusion_sharpe_weight:0.9 → 明确动量偏置
   
3. **两个 Bug**：
   - `predict.py` 和 `score_self.py` 在 `scaler.transform()` 之后从 `processed_df['收盘']` 读取价格计算波动率 → 得到的是 z-score 标准化值而非原始价格
   - `train.py:733` `seq[:, 1]` 取的是 '开盘' 而非 '收盘'（索引应为2）

### 新增函数
- `select_top_stocks_with_gate()` — `utils.py:875-972`
  - 过滤：`predicted_return < min_return_threshold` 不入选
  - 评分：`ranking_score * (1 + predicted_return) / (1 + volatility)`
  - 回退：无股票通过门控时等权选 Top-K

### 模型新增层
- `return_head`: Linear(128→64→1)，预测绝对收益率，Huber loss 回归

## 实验结果

| 版本 | 验证 final_score | 实际5日收益率 | 选中股票 |
|------|-----------------|-------------|---------|
| 官方基准 | 0.038 | **2.52%** | 浙能电力、中国建筑、宁波港、光大银行、中国铁建 |
| V6 (旧) | 0.077 | **0.46%** | 新易盛、中际旭创、福耀玻璃、000568、000725 |
| **V7 (新)** | 0.001 | **6.15%** | 广汽集团(601238)、002709(+19%)、002074(+8%)、000001、002230 |

### 关键指标
- V7 比基准提升 **2.44x**（2.52% → 6.15%）
- V7 比 V6 提升 **13.4x**（0.46% → 6.15%）
- Strategy 3（收益门控）> Strategy 1（等权）> Strategy 2（旧波动率惩罚）
- 与第一名(14.17%)差距：从 -11.65% 缩小到 -8.02%

## 提升因素排序

1. **修复过拟合**（贡献最大）：恢复模型容量 + 辅助任务 + 数据增强，移除动量偏置 → 模型从"只选动量股"变为"综合选股"
2. **收益门控**（边际增量）：return_head 预测绝对收益方向，后处理过滤负收益股票 → +0.13%
3. **Bug 修复**（保证正确性）：波动率计算修复、列索引修复
