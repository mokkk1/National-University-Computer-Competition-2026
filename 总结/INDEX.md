## 按主题浏览

### 模型架构演进
- V6→V7 (06-30): 修复过拟合 + return_head + 收益门控
- V8 五项改进 (07-09): 市场聚合 + 混合标签 + Portfolio Loss
- V9 轻量架构 (07-11): GRU+统计矩，小数据欠拟合
- **最终架构**: V8 Improved（StockTransformer 2.5M + MarketAttentionPooling + MarketGate）

### 后处理策略
- 收益门控 (06-30): predicted_return > 0 过滤
- 市场门控 (07-09): HS300近10日涨跌 → 进攻/防御切换
- **quarter_aware** (07-10): 季末+市场信号联合决策矩阵 🏆

### 数据与训练策略
- Walk-Forward 6窗口 (07-08): 替代固定验证集，避免元过拟合
- 长历史+时间衰减 (07-16): 2010全量但需衰减采样压制旧数据噪声
- 截断实验 (07-19): 2021起最优，旧数据有害

### 特征工程
- 宏观特征 (07-08): 唯一被验证有效的增量特征（W1 +14pp）
- 行业修复 (07-09): AKShare API 失效 → 静态申万映射（1→23类）
- 市场宽度 (07-09): 涨跌比/离散度/偏度/放量信号

### 失败实验（同样重要）
- GBDT Stacking (07-07): -5.32%，中间表征与下游模型不兼容
- 日历特征 (07-10): 模型内部无效，不如后处理
- V9 Lightweight (07-11): 仅W6改善，整体更差
